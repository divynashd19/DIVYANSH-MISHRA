# Divyansh Mishra - Portfolio Website

A modern, tech-forward portfolio website showcasing the work and skills of Divyansh Mishra, a B.Tech Computer Science student specializing in AI/ML and Full Stack Development.

## 🚀 Features

- **Modern Design**: Built with Tailwind CSS and Material Design icons
- **Responsive Layout**: Optimized for all device sizes
- **Dark Theme**: Tech-focused dark color scheme
- **Multiple Pages**: Home, Projects, Education, and About sections
- **Interactive Elements**: Smooth animations and hover effects
- **Contact Form**: Functional contact section

## 🛠️ Tech Stack

- **Frontend**: HTML5, Tailwind CSS
- **Icons**: Google Material Symbols
- **Fonts**: Space Grotesk, Inter
- **Styling**: Custom CSS with CSS Grid and Flexbox

## 📁 Project Structure

```
portfolio/
├── index.html          # Home page with hero, skills, and projects
├── projects.html       # Detailed projects showcase
├── education.html      # Education and certifications
├── about.html          # About page
├── DESIGN.md          # Design system documentation

```

## 🎨 Design System

The design follows a "Tech-Forward Professionalism" approach with:
- Deep nocturnal slate color palette (#0b1326)
- Electric cyan accents (#c3f5ff, #00e5ff)
- Space Grotesk for headlines, Inter for body text
- Tonal layering for depth and hierarchy
